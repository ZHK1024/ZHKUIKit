## Installation

ZHKUIKit is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'ZHKUIKit', :git => 'https://github.com/ZHK1024/ZHKUIKit'
```
